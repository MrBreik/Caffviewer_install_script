https://caff.de/projects/caffviewer/ оффициальный сайт проекта

Данный скрипт распростроняется как есть и за его работоспособность на вашем устройстве я ответственности не несу

При необходимости вы можете доработать его под себя.

Установка программы для просмотра DWG файлов на ОС Astra Linux и AlterOS

Для работы программы требуется рабочая версия Java

Для редактирования параметров запуска caffviewer.jar отредактируйте строчку "Exec=" в CaffViewer.desktop
